About OBT Wallet

  • OBT Wallet is made only for education, experiments, and playful testing.
  • It uses the toy ESS251 elliptic-curve system and the ASH24 toy hash function.
  • Transactions follow a simple P2PK model, where coins are locked to a public key and unlocked by a signature.
  • The blockchain backend is centralized, but the system keeps the transaction data transparent and easy to inspect.
